# OTUS_Python_Project_Shop
Создано веб-приложение интернет магазина на Django.

добавлено виртуальное окружение .venv
выполнена установка django 

Созданы следующие приложения:
-shop_app (главная страница сайта и контактная информация)
-goods_app (информация о категориях и товарах)
-users_app (авторизация, регистрация пользователей)
-basket_app (работа с корзиной)
-orders_app (оформление заказа из корзины)

БД - db.sqlite3
Есть фикстуры, содержащие категории и перечень товаров: fixtures/goods_app
Подгружаем файлы в БД
python manage.py loaddata fixtures/goods_app/cats.json
python manage.py loaddata fixtures/goods_app/prods.json

файл зависимостей - requirements.txt 


В проекте реализована возможность просмотра каталога и карточек товаров неавторизованным пользователям
Можно выполнить поиск товара в графе "Поиск"
Имеется возможность зарегистрироваться, авторизоваться, выйти (вход настроен по почте)
права пользователя настраивает суперпользователь
обычный пользователь может войти в личный кабинет, добавить товар в корзину, оформить заказ и просмотреть свои заказы
персонал имеет те же возможности, но еще есть возможность работы с продуктами через сайт: Каталог - Все товары - Управление товарами.  CRUD (добавить, изменить, удалить)
Также для персонала есть возможность перейти с сайта в админн-панель с возможностью просмотра и добавления категорий и полные права для товаров (в моем исполнении)

При оформлении заказа результат зависит от баланса пользователя - успешный заказ при достаточном балансе. Баланс нового пользователя по умолчанию = 100 000. 
Баланс можно изменять на сайте в "Мой профиль - личный кабинет"
